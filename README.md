# Today OS

A minimal operating system for your day.

This project integrates:

- **Today** – your task & notes manager  
- **Time Tool** – your time-focused calendar/linear schedule  

All in a **clean, ultra-minimal, mobile-friendly interface**.

---

## Live Demo

Visit the landing page:

[Today OS Live](https://pprashil26.github.io/today-os/)

- **Open Today:** Click “Open Today” → or use `1` keyboard shortcut  
- **Open Time Tool:** Click “Open Time Tool” → or use `2` keyboard shortcut  
- **Mobile Swipe:** Swipe left/right to switch between tools
